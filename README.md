Secure Food Delivery Web Application

A cybersecurity-focused food delivery web application built using Flask.

Features

- User Registration & Login
- Password Hashing using bcrypt
- Multi-Factor Authentication (OTP)
- Intrusion Detection System (Brute Force Detection)
- Account Locking after suspicious activity
- Role-Based Access Control (RBAC)
- AES Data Encryption
- Session Timeout Protection
- Security Activity Logs
- Modern UI with secure indicators

Security Implementations

- Encrypted user data storage
- Brute-force login detection
- OTP verification system
- Secure session handling
- Protection against unauthorized access
- Secure logout and timeout system

Technologies Used

- Python (Flask)
- SQLite
- HTML / CSS / JavaScript
- bcrypt
- cryptography (Fernet)

Purpose

This project demonstrates how real-world food delivery platforms can implement advanced cybersecurity mechanisms to protect user data and prevent attacks.
